Como o sistema decide aceitar?

Transparente e inteligente

1

Lojista anuncia 10 TVs com preço mínimo de R$ 2.000 (secreto)

Você vê apenas: "Preço sugerido: R$ 2.500"

2

Cliente A oferece R$ 2.200 → ACEITA

Média = R$ 2.200 > R$ 2.000 ✓

3

Cliente B oferece R$ 1.900 → ACEITA

Média = (2.200 + 1.900) / 2 = R$ 2.050 > R$ 2.000 ✓

4

Cliente C oferece R$ 1.900 → RECUSADA

Média = (2.200 + 1.900 + 1.900) / 3 = R$ 2.000 — não ultrapassa o mínimo ✗
